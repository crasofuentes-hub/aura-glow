# Code of Conduct

We are committed to a professional, harassment-free experience for everyone.

## Expected Behavior
- Be respectful, constructive, and precise.
- Assume good intent; focus on technical outcomes.
- Disclose conflicts of interest where relevant.

## Unacceptable Behavior
- Harassment, discrimination, threats, or personal attacks
- Sharing private information (doxxing)
- Posting secrets, tokens, or sensitive user data

## Enforcement
Project maintainers may remove or edit content and restrict participation for violations.

## Reporting
Report incidents to the maintainer via GitHub profile: **crasofuentes-hub**.

# Quickstart (Windows)

## Prereqs
- Git
- PowerShell 7 (pwsh)
- CMake (for engine demo)
- (Optional) Flutter SDK for app/

## Run verification
pwsh -File .\scripts\verify.ps1 -Strict

## Build & run engine demo
pwsh -File .\scripts\demo.ps1

If both commands return OK, the repository scaffold is healthy.
